Catapult, custom model created to play in Warcraft 3 custom games.

This model was maded intend to be similar to Nikita Bufatin - Concept Artwork, from this i started working with WoW resources to composite it.

All textures are with the better quality as possible for cinematic purposes, so it's up to the user to optimize it or not.

This Model contain the following animation sequences.

  1. Attack
  2. Death
  3. Decay
  4. Portrait
  5. Stand
  6. Walk
