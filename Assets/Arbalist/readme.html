Description

So... yeah, there's apparently not a single crossbowman in Warcraft 3. I thought to myself: well, maybe the riflemen could be a nice base model. And boy, it was.

Changed the height, the weapon, every single animation (ugh), the collision spheres, the geosets... It has TC, an icon, a working attack animation.

Has no custom textures.


So, to import this model, make sure to set these values (both violet and black):

Black values are the same as the archer. You can base the unit with the archer model to save some time.

Also, he has 2 different attack animations: Attack One (against ground units) and Attack Two (against air units).




Known bugs:
-He has the random blood bug from many imported units that had the .mdl changed. It's not gamebreaking by any means (when you pick an imported unit, a blood spout forms in the center of the map for no reason at all).
