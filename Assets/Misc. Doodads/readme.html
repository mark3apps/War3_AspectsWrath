I'm putting together some doodads that I plan to use on a project and figured I'd share as I create new stuff. I have a few usable doodads that are farm themed, so I'm starting by sharing that, but I have some other stuff I'm working on too. Also if anyone has any doodad requests, if it's simple enough and doesn't require a whole lot of animating, let me know and maybe I'll add it to this package.

I also have dozens of basic doodads I've ripped from other doodads/units/buildings, like swords and shields to add for decorative purposes. If anyone would like me to upload what I've got, I'll try to make some time to do that. I'd just like to add stuff that people will actually find useful, because some of the stuff I have might not be super high quality/working correctly.
