Description

"From the frozen land to the far north, humans fight an endless conflict with the dark. With few months of daylight, this battle seems a futile one. But against these odds, there thrives a culture of powerful men and women one that will see the dawn break once more"


A simple collection of axe-wielding units for my campaign. Contains the Yarl hero model, the Stormbjorn model with and without hero glow and the axe-wielding warrior unit model.

Yarl: Duel wielding axe warrior has 2 extra animations a secondary attack and a VERY janky sleep animation I use for cinematic purposes.

Stormbjorn Captain: A simple edit of the Stormbjorn uses jailor Kassan head and matches the icon very well. Wields a giant poleaxe. - Comes with and without hero glow but no decay animation. If someone knows how to using Retera's model studio please DM me and I will update.

Stormbjorn: Simple elite unit wielding a large pole-axe again, fits nicely with the captain model. - Comes with and without hero glow but no decay animation. Again if someone DM's me I will update.

Warrior: Simple axe-wielding basic melee unit nothing special
